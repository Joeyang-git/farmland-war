# 二、核心玩法概述

本游戏为单机对称对战类游戏（适配移动端+PC端），核心围绕「二维网格地块占领」与「动物技能对抗」展开，第一版优先实现单机玩法，支持玩家与多电脑玩家对战。玩家从初始主基地出发（出生点位按关卡预设或随机生成），沿己方领地边界逐步扩张，动物建筑在满足规则时**自动释放技能**，攻击敌方建筑、占领无主地块/废墟；玩家可在地图上放置**最多一面攻略旗**标记高优先级方向，系统在为自动技能选址时优先向旗标区域倾斜。胜利目标为摧毁敌方主基地或使敌方破产。核心亮点为「沿边界扩张」「包围占领收益」「封锁战术压制」与「少操作、意图指挥」，新增自定义关卡与遭遇战模式，规则简洁、策略性强，适配双平台操作习惯。

# 三、核心规则（全部定稿，适配单机+多电脑玩家）

## 3.1 地图与地形规则

1. 地图为「二维网格」，玩家与电脑玩家主基地出生点位**不固定对称**，可根据关卡预设或随机生成；自定义关卡可手动设置所有玩家（玩家+电脑）的出生点位与初始领地大小，遭遇战模式中出生点位随机生成，确保玩家与电脑玩家初始条件公平（初始资金、初始领地范围一致）。

2. 地形随机生成3类元素，均不可重叠，规则如下：

3. 无主地：1点耐久，可被攻击占领，为核心扩张目标。

4. 废墟：2点耐久，可被攻击占领；若被包围，可自动归包围方（无需攻击）。

5. 障碍地形（树林/河流/池塘）：不可占领、不可建造、不可通行，不参与包围判定，随机分布形成天然路口；自定义关卡可手动调整障碍位置。

6. 地块规则：一格一地，不可重叠，地块归属以「最后一击」或「包围判定」为准。

## 3.2 扩张规则（核心）

1. 所有地块（无主地/废墟）只能「沿己方已有领地的相邻格」向外扩张，不可隔空、不可跳占（玩家与电脑玩家通用此规则）。

2. 占领方式：攻击无主地/废墟，打掉全部耐久后，地块归攻击方所有；建筑被摧毁后，其占地块归摧毁方所有。

3. 玩家/电脑玩家用己方领地形成「完整封闭圈」（无缺口、不依赖障碍，仅靠己方领地闭环），圈内部所有无主地自动归该方所有（无需攻击）。

4. 若圈内有敌方（玩家/电脑玩家）建筑，建筑及所属地块不自动归属，需攻击摧毁后才能占领。

## 3.3 动物生产与建筑建造规则

1. 购买动物等于建设动物建筑，所有建筑「一次性付费，无持续消耗」（玩家与电脑玩家通用）。

2. 建造规则：只能建造在「己方完全拥有的地块」上，建筑所占格子必须完全属于己方，不可跨边界。

3. 建筑尺寸：按动物类型设定，统一规范如下（玩家与电脑玩家共用同一规则）：

4. 低级动物（1×1）

5. 中级动物（2×2）

6. 高级动物（3×3）

## 3.4 操作与技能自动释放规则（适配双平台）

1. **技能全部自动**：所有动物技能（无论主动/被动）均由系统在 CD 就绪后**自动执行**，玩家无需手动点选目标或拖拽。产钱类按固定周期产出，扩张/攻击类按 CD 周期自动选点施放。

2. **选点默认策略**：无旗时，系统按「己方领地邻接格」中**价值最高**的格（优先：敌方建筑 > 废墟 > 无主地）随机选一个合法目标。无合法目标时本次不触发，CD 重置继续等待。

3. **攻略旗（唯一）**：每位真实玩家同时最多**一面攻略旗**，插在地图上某个格子，表示「我想往这里打」。点击地图插旗，在其他位置再点则**移动**旗（始终只有一面），长按/右键可**拔旗**。不可插在障碍格。

4. **旗对选点的影响**：存在攻略旗时，系统在合法目标中优先选取**距旗更近**的格，使动物自动向旗的方向扩张和攻击；旗拔掉后恢复默认策略。

5. **合法施放约束**：技能目标必须是**与己方领地相邻的格**，不可隔空；若无任何合法格则等待，不空放。

6. **CD 规则**：封锁状态下 CD 停止推进；正在 CD 中的建筑不参与当次施放。伤害均为固定值，无暴击，建筑无自动反击。

## 3.5 经济系统规则

1. 产钱来源：产钱动物被动技能产钱，被摧毁后立刻停止产钱（玩家与电脑玩家通用）。

2. 初始资金：游戏开始时，玩家与每位电脑玩家获得固定初始资金（具体数值后续配置，所有玩家初始资金一致）。

3. 金钱规则：金钱最低为0，不会出现负数；钱不够时，无法建造任何动物建筑；无金钱上限（玩家与电脑玩家通用）。

## 3.6 主基地规则

1. 初始设定：玩家与电脑玩家主基地出生点位按关卡预设或随机生成，不固定对称；主基地占地2×2，不可移动、不可重建、不可升级。

2. 属性：无技能、不产钱，仅作为核心胜利目标；耐久建议30点（比高级建筑更肉，具体数值后续配置，玩家与电脑玩家主基地耐久一致）。

3. 失败判定：主基地被摧毁，该方（玩家/电脑玩家）立刻战败，退出本局游戏。

## 3.7 包围封锁规则

1. 封锁触发条件：敌方（玩家/电脑玩家）建筑/地块被己方领地完全闭环包围，且该建筑/地块无法通过陆地连通到敌方主基地（形成孤岛）。

2. 封锁效果：技能停止充能（CD不再推进）；主动/被动技能无法释放

3. 建筑与地块归属不变，需攻击摧毁后才能易主（玩家与电脑玩家通用此效果）。

4. 例外情况：若敌方（玩家/电脑玩家）建筑与基地有连接，即使被己方围一圈，也不触发封锁（仅限制敌方扩张，敌方内部正常运作）。

## 3.8 胜负条件（定稿，适配多电脑玩家）

1. 核心胜负：存活到最后一方（玩家/电脑玩家）获胜；一方主基地被摧毁，该方立刻战败。

2. 次要胜负（破产）：一方无产钱建筑（鸭、猪全部被摧毁）且金钱=0，持续10秒，该方战败（玩家与电脑玩家通用）。

3. 无超时设定，避免膀胱局。

# 四、开发优先级（第一版，单机优先）

## 4.1 核心必做（上线基础，单机为主）

1. 地图生成：对称网格、随机地形（无主地/废墟/障碍），支持自定义关卡的地图编辑功能。

2. 扩张与占领：沿边界扩张、地块耐久计算、包围占领判定（玩家与电脑玩家通用）。

3. 建筑系统：所有确认动物建筑（产钱：鸭、鹅、猪；战斗：鸡、牛、狗、马、孔雀、大象、老虎）的建造、尺寸限制、一次性付费逻辑，分类清晰可执行。

4. 技能系统：所有动物技能的自动释放、攻略旗（单面）加权选点、CD 计算、固定伤害、范围判定，适配双平台操作。

5. 经济系统：初始资金、鸭/猪自动产钱、金钱无负逻辑（玩家与电脑玩家通用）。

6. 主基地：固定位置、耐久设定、被摧毁判定（玩家与电脑玩家通用）。

7. 胜负判定：主基地摧毁、破产判定（10秒计时），适配多电脑玩家存活判定。

8. 单机模式：遭遇战模式（多电脑玩家支持）、自定义关卡模式（地图编辑、关卡保存/加载）。

9. AI逻辑：基础电脑玩家AI（扩张、造建筑、攻击），支持1-3名电脑玩家同时对战。

10. 双平台适配：移动端与PC端操作适配，保证玩法一致。
